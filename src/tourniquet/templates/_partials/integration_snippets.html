Use this key in your tools Show env-var snippets for Claude Code, Cursor, Python, Node, curl, PowerShell

Set these two env vars and any Anthropic-compatible client will route through Tourniquet on this machine. Replace tq_YOUR_TOKEN with the token you saved when you created or last rotated this key. Lost your token? Hit Rotate in the Control panel above to mint a fresh one — the old one stops working immediately.

Claude Code (bash / zsh) 
export ANTHROPIC_BASE_URL={{ proxy_url }}
export ANTHROPIC_API_KEY=tq_YOUR_TOKEN
claude

Add the two export lines to your ~/.zshrc or ~/.bashrc to persist across shells.

Cursor (any platform) 
ANTHROPIC_BASE_URL={{ proxy_url }}
ANTHROPIC_API_KEY=tq_YOUR_TOKEN

Cursor Settings → Cursor Settings → Models → set Custom Anthropic API Key. Or set both as environment variables before launching Cursor.

Python SDK (anthropic) 
import anthropic

client = anthropic.Anthropic(
    api_key="tq_YOUR_TOKEN",
    base_url="{{ proxy_url }}",
)

resp = client.messages.create(
    model="claude-haiku-4-5-20251001",
    max_tokens=100,
    messages=[{"role": "user", "content": "hi"}],
)
print(resp.content[0].text)
Node SDK (@anthropic-ai/sdk) 
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: "tq_YOUR_TOKEN",
  baseURL: "{{ proxy_url }}",
});

const resp = await client.messages.create({
  model: "claude-haiku-4-5-20251001",
  max_tokens: 100,
  messages: [{ role: "user", content: "hi" }],
});
console.log(resp.content[0].text);
curl (smoke test, ~1 cent)

  -H "content-type: application/json" \
  -H "anthropic-version: 2023-06-01" \
  -d '{"model":"claude-haiku-4-5-20251001","max_tokens":50,"messages":[{"role":"user","content":"say hi in 5 words"}]}'
PowerShell (Windows) 
$env:ANTHROPIC_BASE_URL = "{{ proxy_url }}"
$env:ANTHROPIC_API_KEY  = "tq_YOUR_TOKEN"

Add to your $PROFILE file to persist across sessions. Revert with Remove-Item Env:ANTHROPIC_BASE_URL, Env:ANTHROPIC_API_KEY.

cmd.exe (Windows) 
set ANTHROPIC_BASE_URL={{ proxy_url }}
set ANTHROPIC_API_KEY=tq_YOUR_TOKEN

Use setx instead of set to persist permanently.

Safety net: if Tourniquet ever stops, unset the env vars and your agent falls back to direct Anthropic with whatever key you originally had set.
bash/zsh: unset ANTHROPIC_BASE_URL ANTHROPIC_API_KEY
PowerShell: Remove-Item Env:ANTHROPIC_BASE_URL, Env:ANTHROPIC_API_KEY
