Keep Tourniquet running while this PC sleeps

Windows suspends idle machines by default, which freezes Tourniquet's cap enforcement. Launch the proxy from PowerShell with a system-execution-state request so Windows keeps it awake:

powercfg /requestsoverride PROCESS tourniquet.exe SYSTEM
tourniquet start

To revoke the override later, run powercfg /requestsoverride PROCESS tourniquet.exe with no flags. Verify what's keeping the system awake any time with powercfg /requests. For a 24/7 setup, see the deployment guide — running as a Windows service avoids the sleep problem entirely.
