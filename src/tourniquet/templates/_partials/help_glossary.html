Glossary — what these words mean

sk-ant- key
Your real Anthropic API key — the secret string from console.anthropic.com that authorises you to use Claude. Tourniquet stores it encrypted; your agents never see it directly.
tq_ token
A proxy token Tourniquet generates for each key. Your agents use this instead of the real sk-ant- key. Tourniquet swaps it for the real key before forwarding to Anthropic, so your actual key never leaves this machine. Shown once on creation/rotation — Tourniquet stores only the bcrypt hash, so the plaintext can't be recovered. If you lose your copy, use the Rotate button in the Control panel to generate a fresh one (the old one stops working immediately).
daily_cap
The maximum dollars Tourniquet will let this key spend in one calendar day (midnight UTC to midnight UTC). Once reached, requests are blocked until tomorrow — unless you lift the cap.
lifted_cap
A temporary higher cap that overrides the daily cap for the rest of today only. It expires automatically at midnight UTC. Your original daily cap comes back tomorrow — you don't need to re-enter it.
kill switch / kill_enabled
When on (the default), Tourniquet actually blocks requests once the cap is hit. When off, it only records spend and sends alerts — nothing is blocked. Leave it on unless you have a specific reason to disable it.
kill now
Immediately clamps today's remaining budget to zero (or to what you've already spent). Only affects today — tomorrow's cap is unchanged. Use this to stop a runaway agent mid-flight.
auto_tune
Tourniquet can look at your recent spending history and suggest (or automatically adjust) a better cap. "off" = no suggestions, "suggest" = shows a recommendation you can accept or ignore, "creep" = quietly adjusts the cap over time.
profile
A preset that controls how Tourniquet behaves near and at the cap. "standard" blocks at 100%; "monitor" only alerts (no blocking); "aggressive" starts limiting earlier. Standard is the right choice for most people.
ceiling / absolute_ceiling
The hard upper limit for this key — auto-tune AND the "🚀 ceiling" lift button can never push the cap above it. A safety net so a runaway auto-tune or stray tap can't drive spending to infinity. Set initially from ABSOLUTE_CEILING_USD_CENTS in ~/.tourniquet/.env when the key is created (default $100/day). Edit per-key via the Ceiling input in the Control panel; it must always be ≥ daily cap.
monitor mode
Running with the "monitor" profile (or kill switch disabled). Tourniquet watches and alerts but never blocks requests. Useful for understanding your usage before you commit to hard limits.
midnight UTC
The point in time when the daily cap resets. UTC is a fixed time zone — it doesn't change with daylight saving. If you're in London in summer, midnight UTC is 1:00 AM for you. In New York, it's 8:00 PM the previous evening.
