Keep Tourniquet running while this Mac sleeps

By default macOS sleeps after a few minutes of inactivity, which suspends Tourniquet. Run it with caffeinate instead to block sleep while the proxy is up:

caffeinate -di tourniquet start

The screen can still dim — only sleep is blocked. Stop the proxy with Ctrl+C in that terminal, or pkill caffeinate from another. To keep going without a laptop awake at all, see the 24/7 deployment guide for Pi / Proxmox / Docker / cloud VM.
